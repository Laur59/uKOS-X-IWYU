Ozone uKOS-X RTOS Plugin
=========================

(c) 2025-2026, Edo. Franzi, 2025-01-01



This plugin enabled uKOS-X RTOS awareness in SEGGER Ozone debugger.

1. **uKOS.js: that uses Ozone SmartView display facilities in multi window.**

Be sure that the latest Java package is installed.

Installation
------------

1. Lauch Ozone

2. Enable the plugin:

   1. **Enabling the SmartView plugin:** in Ozone consol type: Project.SetSmartViewPlugin("xyz/OS_Kernel-X/Tools/Ozone_Plugin/uKOS-X"). In this mode, it is possible to open many instances of the plugin.

3. Add view from menu View/SmartView.
